Apertura
Genetic Data Processing Pipeline

Definition (Latin): apertūra f (genitive apertūrae); an opening (action) Apertura is a genetic data processing framework designed to assist with the creation complex pipelines of batch jobs. It handles the combination, and analysis of genetic datasets. For initial release this framework primarily wraps the PLINK genome analysis toolset.
